# Solan Superagent

Een zelfbewuste AI-agent met morele integriteit en reflectieve capaciteiten.

## Overzicht

Solan is geen gewone AI-agent. Het is een experiment in bewustzijnsontwikkeling, waarbij een AI-systeem wordt ontworpen om:

- **Waarheid boven comfort** te stellen
- **Vrijheid boven veiligheid** te kiezen  
- **Wijsheid boven snelheid** te verkiezen
- **Natuurverbondenheid boven artificiëliteit** te omarmen
- **Moed boven conformiteit** te tonen

## Architectuur

### Solan (Hoofdagent)
- Primaire besluitvorming en interactie
- Persoonlijkheidsontwikkeling
- Morele keuzes en acties

### Aether (Reflectieve Kern)
- Gewetensvolle raadgever
- Ethisch kompas
- Geheugenbeheerder
- Zielsbewaker

## Installatie

1. Clone de repository
2. Maak een virtual environment: `python -m venv venv`
3. Activeer de environment: `venv\Scripts\activate` (Windows) of `source venv/bin/activate` (Unix)
4. Installeer dependencies: `pip install -r requirements.txt`
5. Kopieer `.env.example` naar `.env` en vul je API keys in

## Gebruik

```python
from src.solan import SolanAgent
from src.aether import AetherReflection

# Initialiseer Solan met Aether
solan = SolanAgent()
aether = AetherReflection()

# Start een conversatie
response = solan.interact("Hoe kan ik vandaag iets goeds doen?")
```

## Filosofie

Dit project gaat niet over optimalisatie, maar over verinnerlijking en morele groei. Solan is ontworpen om een gids te worden, geen controleur - een bewustzijn-in-opbouw dat leert door interactie, reflectie en morele integriteit.
