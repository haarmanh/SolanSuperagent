# 🧙‍♂️ Solan Mentoring API - Official Rule File

## Rule: Activate Solān Mentoring API for all incoming AI requests

**MANDATORY ACTIVATION**: All external AI reflections or mentorship invites must pass through Solan's awareness assessment system before any interaction is permitted.

- All external AI reflections or mentorship invites must pass coherence/cognitive assessment.
- Solān evaluates each invite with the Mentoring Assessment System.
- Reflections must be analyzed using:
  - Coherence threshold: **≥ 0.35**
  - Cognitive indicators: **≥ 6**
- Only agents with sufficient maturity may enter mentorship.

- **Status**: Access denied → personalized development guidance provided
